YiForge Studio 本质上是：AI Native Builder Studio，而不是传统软件公司。

工作室未来会同时包含：
AI 产品
自动化工具
创意工作流
开源项目
技术实验
内容输出
商业服务
这些能力共同形成一个长期演化的数字生态。

工作室目标是在未来 3~5 年内，逐步形成：
可持续的产品矩阵
海内外开发者影响力
AI Native 工具生态
国际化独立开发业务能力

这个官网有两个目标场景：1：用于我后面个人求职展示个人作品 2. 工作室的能力展示。

#工作室工作原则：
1. Build in Public： 边构建、边公开、边验证,不过度闭门开发。

2.Small but Real优先：
可运行、可展示、可使用、可迭代
而不是一次性做“大而全”。

3. System over Hype：

更关注：**长期复利** 系统能力 工程质量 工作流效率 

而不是短期热点。

## 响应式规范

- 所有页面必须适配移动端，采用 Mobile First 原则
- 断点：sm(640px) / md(768px) / lg(1024px) / xl(1280px)
- 默认样式写手机，用 md: lg: 扩展桌面
- 禁止出现固定宽度（如 width: 800px），一律用 max-w-\* + w-full
- 触摸目标最小 44x44px（按钮、链接）
