# 需求
我想建立要给 AI 开发工作室的官网。
名字：AIForge Studio 工作室官网

# 官网目录
1. 首页
    首页
    ├── Hero区          ← 第一眼
    ├── 服务简介         ← 我能做什么
    ├── 作品预览         ← 我做过什么
    ├── 开发流程简介      ← 怎么合作
    └── 联系入口         ← 怎么找我
2. 服务页，放详细的服务介绍和开发流程：
3. 作品集,后续有项目就往里面添加，写项目介绍。
4. 博客（Blog）
   ├── AI 思考
   ├── 技术总结
   └── 折腾记录
5. 关于我，简历和联系方式
   ├── 个人简介
   ├── 技能栈
   ├── 工作经历
   └── 联系方式

# 技术需求
1. 支持 SEO 友好
2. 支持中英的架构。但是现在只需要中文的版就行
3. 支持明暗模式
4. 要苹果设计风格，简约。


# 技术栈
核心框架：

Next.js 14（App Router）— 前端框架
TypeScript — 类型安全
React 18 — UI 库

样式：

Tailwind CSS — 样式框架
Shadcn/ui — 组件库
next-themes — 亮暗模式

工程化：

Turborepo — Monorepo 管理
pnpm — 包管理器
ESLint — 代码规范
Prettier — 代码格式化

内容管理：

MDX — 博客文章（Markdown + React组件）
Contentlayer — MDX 文件管理

SEO：

next/metadata — 页面 SEO
next-sitemap — 自动生成 sitemap
Open Graph — 社交分享预览

动画：

Framer Motion — 页面动画效果

部署：

GitHub — 代码托管
Cloudflare Pages — 免费部署
自定义域名 — aiforge.studio 或 aiforge.dev

以后扩展（现在不用）：

NestJS — 后端框架
Prisma — 数据库 ORM
PostgreSQL — 数据库
Taro — 小程序框架
