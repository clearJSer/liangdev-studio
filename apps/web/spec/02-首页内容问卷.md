# 首页内容问卷

## 文档职责

这份文档用于收集 YiForge Studio 首页内容所需的真实素材。

你不需要一次性想清楚，也不需要写成漂亮文案。可以用关键词、短句、项目名、草稿想法来回答。

后续 AI 会基于这些回答，继续整理出：

- 首页定位
- 首页区块结构
- 首页中文文案
- 项目展示素材
- 暂不确定的问题
- 后续英文版内容基础

## 填写原则

- 先写真实想法，不追求完整。
- 不确定的地方可以直接写“不确定”。
- 如果一个问题暂时没有答案，可以留空。
- 可以反复补充和修改。
- 不要为了显得成熟而写企业化话术。
