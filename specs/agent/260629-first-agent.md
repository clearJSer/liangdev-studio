# 背景
经过与Claude Code和CodeX做了多轮的探讨。我决定这个工作室做的第一个Agent是coding Agent。
理由是，Coding Agent是最成熟的落地场景，非常的完善和能锻炼技术。
因为它也比较难，也比较对我的实际场景中开发有帮助，所以我做这个应用的开发，有助于我去收集这方面的一些学习资料和思考。我觉得对我来说是比较收益比较高的一件事情。
三是这个也是我选好的第一个Agent，有了这个Agent的开发经验之后，再开发其他Agent可能也会有一些经验的沉淀吧。
选过来选过去，纠结再三，都是在思考，没有行动，所以直接敲定下来，做一个Coding Agent。
无论他做的成功与失败，或者效果好与坏，先行动，先干起来。
啊，今天晚上也看了一些AI的视频，说有了AI之后，最宝贵的就是行动力。行动力并不意味着成功，但是它是成功或不可或缺的一种行为。
